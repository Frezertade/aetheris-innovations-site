# Online Presence — Monthly Management SOP

## Week 1: Performance Review
- [ ] Pull GA4 traffic report
- [ ] Check Google Search Console impressions/clicks
- [ ] Review chat conversations + lead quality
- [ ] Check GBP insights (calls, direction requests)
- [ ] Compile monthly report from `templates/monthly-report.md`

## Week 2: Optimization
- [ ] Review and update 2-3 meta descriptions if CTR is low
- [ ] Add or refresh 1 location page (if applicable)
- [ ] Check site speed (PageSpeed Insights)
- [ ] Fix any broken links or errors

## Week 3: Content
- [ ] Publish 1 blog post (Growth/Authority)
- [ ] Update homepage or key page if needed
- [ ] Review chat logs for new FAQ/training opportunities
- [ ] Post GBP update (photo, offer, or event)

## Week 4: Reporting + Planning
- [ ] Finalize monthly report
- [ ] Send to client
- [ ] Propose next month priorities
- [ ] Invoice monthly fee

## Quarterly (every 3 months)
- [ ] Full SEO audit
- [ ] Competitor review
- [ ] Design refresh discussion
- [ ] Upsell opportunity assessment
