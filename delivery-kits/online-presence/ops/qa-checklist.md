# Online Presence — QA Checklist

Run this before every launch.

## Functionality
- [ ] All navigation links work
- [ ] All buttons are clickable
- [ ] Contact form submits successfully
- [ ] Form notifications arrive in inbox
- [ ] Chat widget loads and responds
- [ ] Phone numbers are clickable (tel: links)
- [ ] Email links work (mailto:)

## Responsive
- [ ] iPhone (375px)
- [ ] iPad (768px)
- [ ] Desktop (1440px)
- [ ] No horizontal scroll on mobile
- [ ] Text is readable without zooming

## Performance
- [ ] Page loads in < 3 seconds
- [ ] Images are optimized (< 200KB each)
- [ ] No console errors
- [ ] Core Web Vitals passing

## SEO
- [ ] Title tags unique and descriptive
- [ ] Meta descriptions present
- [ ] H1 tag on every page
- [ ] Alt text on all images
- [ ] Schema markup (LocalBusiness)
- [ ] Sitemap.xml submitted
- [ ] Robots.txt present

## Legal
- [ ] Privacy policy page
- [ ] Terms of service page
- [ ] Cookie notice (if required)

## Analytics
- [ ] GA4 tracking code installed
- [ ] Google Search Console verified
- [ ] Goals/events configured
