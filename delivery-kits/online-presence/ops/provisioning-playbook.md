# Online Presence — Provisioning Playbook

## Starter Presence ($499 setup, $99/mo)

### Technical Setup
- Domain: Use existing or register new ($12-15/yr via Cloudflare)
- Hosting: Vercel (free) or Cloudflare Pages (free)
- CMS: Static HTML or lightweight builder
- Analytics: Google Analytics 4 + Google Search Console

### Build Checklist
- [ ] 1-page conversion site (hero, services, about, contact)
- [ ] Mobile-first responsive
- [ ] Contact form → email notification
- [ ] Google Business Profile claimed + optimized
- [ ] Basic meta tags + schema markup
- [ ] Privacy policy + terms pages

### Handoff
- Domain credentials
- GA4 access
- GBP login info

---

## Growth Presence ($1,500 setup, $199/mo)

### Technical Setup
- Domain: As above
- Hosting: Vercel Pro ($20/mo) or Cloudflare Pages
- CMS: Next.js or equivalent
- Analytics: GA4 + GSC + Meta Pixel (optional)

### Build Checklist
- [ ] 5-page site (home, services, about, testimonials, contact)
- [ ] Aetheris Chat widget installed
- [ ] Chat trained on business content
- [ ] Lead capture form → email + CRM
- [ ] Local SEO: location pages, schema, citations
- [ ] Review generation system
- [ ] 30-day content calendar delivered

### Integrations
- Aetheris Chat API key
- Email routing (Zapier or direct)
- CRM: HubSpot free or GoHighLevel

### Handoff
- All Starter items
- Chat dashboard access
- CRM login
- Content calendar

---

## Authority Presence ($3,500 setup, $499/mo)

### Technical Setup
- Domain: As above
- Hosting: Vercel Pro or dedicated
- CMS: Next.js with dynamic routing
- Analytics: Full stack (GA4, GSC, Meta, ad tracking)

### Build Checklist
- [ ] Full multi-page website + landing pages
- [ ] Aetheris Chat + voice call integration
- [ ] Lead automation + CRM pipeline
- [ ] SEO content plan + 5 blog posts
- [ ] Google Ads / Meta Ads tracking pixels
- [ ] Dedicated Slack channel
- [ ] Monthly strategy calls

### Integrations
- All Growth items
- Voice provider (Bland AI or similar)
- Advanced CRM automation
- Ad platform accounts

### Handoff
- All Growth items
- Slack channel invite
- Ad account access
- 90-day growth roadmap
