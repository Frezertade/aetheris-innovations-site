# Online Presence — 7-Day Delivery SOP

## Day 1: Kickoff + Discovery
- [ ] Confirm payment received
- [ ] Send welcome email with intake questionnaire
- [ ] Schedule kickoff call (if not done)
- [ ] Collect: logo, brand colors, existing content, photos
- [ ] Set up project folder in Google Drive / Notion

## Day 2: Structure + Copy
- [ ] Write sitemap / page structure
- [ ] Draft all page copy (headlines, body, CTAs)
- [ ] Select or create wireframe
- [ ] Send copy to client for approval

## Day 3: Design
- [ ] Build homepage design
- [ ] Build interior pages
- [ ] Mobile responsive check
- [ ] Internal review

## Day 4: Development
- [ ] Code / build all pages
- [ ] Install Aetheris Chat (Growth/Authority)
- [ ] Set up forms + lead routing
- [ ] Configure analytics

## Day 5: Content + SEO
- [ ] Populate all content
- [ ] Optimize meta titles, descriptions
- [ ] Add schema markup
- [ ] Set up Google Business Profile (Starter)
- [ ] Create location pages (Growth/Authority)

## Day 6: QA + Revisions
- [ ] Run QA checklist
- [ ] Test all forms and links
- [ ] Test mobile, tablet, desktop
- [ ] Test chat functionality
- [ ] Send preview link to client
- [ ] Collect feedback, make revisions

## Day 7: Launch
- [ ] Deploy to production domain
- [ ] Final QA on live site
- [ ] Submit sitemap to Google
- [ ] Send handoff document
- [ ] Schedule 30-day check-in

## Extended Timeline (Growth/Authority)

| Tier | Extra Days | Focus |
|------|-----------|-------|
| Growth | +3 days | Chat training, CRM setup, content calendar |
| Authority | +7 days | Voice integration, ad tracking, blog content, Slack setup |
