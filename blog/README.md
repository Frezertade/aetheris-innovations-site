# Automated Blog System

This folder contains an automated blog post generator for Humanized AI and automation topics. Run `node generateBlog.js` daily to create a new SEO-optimized blog post in HTML format.

## Features
- Daily blog post generation
- SEO meta descriptions and keywords
- Schema markup
- Alt text for images
- Natural keyword placement

## Usage
1. Ensure Node.js is installed.
2. Run `node generateBlog.js` to generate today's blog post.
3. Deploy the generated HTML file to `/blog/`.

## Customization
- Update keywords in `generateBlog.js` for more topics.
- Replace `/public/ai-automation.jpg` with your own images for better branding.
